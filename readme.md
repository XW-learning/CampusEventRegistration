项目六：校园活动报名系统

项目描述：

主要用于学校各类校园活动的发布以及学生报名参与，方便活动组织者管理报名情况。

功能模块：

1.活动发布：填写活动名称、时间、地点、详情等信息并保存。

2.活动展示：学生端展示未截止报名的活动列表及详情。

3.报名功能：学生报名活动，系统记录报名信息（关联学生与活动）。

4.报名情况查询：活动组织者查看各活动的报名学生名单。

5.多条件筛选：学生按活动类别、时间范围、地点及关键字组合筛选活动。

6.报名审核：学生报名后需经审核，审核通过方为报名成功。

7.活动签到与统计：现场签到通过数字签到码记录签到状态，活动后统计实际参与、迟到人数等。
